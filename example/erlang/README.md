# AtomVM Erlang Example Programs

Welcome to the AtomVM Erlang Example programs.

The following applications demonstrate various peripherals and features on the ESP32 platform:

| Application | Description |
|-------------|-------------|
| [control_led](control_led/README.md) | Control an LED via a web interface |
| [encoder](encoder/README.md) | Rotary encoder example |
| [esp32_heart](esp32_heart/README.md) | LED matrix heart display |
| [gpio_interrupt](gpio_interrupt/README.md) | GPIO interrupt handling |
| [hello_world](hello_world/README.md) | Minimal hello world application |
| [joystick](joystick/README.md) | Joystick ADC input |
| [ledc](ledc/README.md) | LED PWM controller with web control |
| [mpu6500](mpu6500/README.md) | MPU6500 accelerometer/gyroscope sensor |

For build and flash instructions, please refer to the [root README](../../README.md).
