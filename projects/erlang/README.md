# AtomVM Erlang Project Programs

Welcome to the AtomVM Erlang Project programs.

The following applications are larger projects demonstrating complete use cases on the ESP32 platform:

| Application | Description |
|-------------|-------------|
| [block_breaker_2led](block_breaker_2led/README.md) | Block breaker game on 2 LED matrices |
| [calculator](calculator/README.md) | Calculator with LCD and keypad |
| [car_project](car_project/README.md) | WiFi-controlled car |
| [hour_glass](hour_glass/README.md) | Hour glass game with motion sensing |
| [self_balance_robot](self_balance_robot/README.md) | Self-balancing robot |
| [snake_blockbreaker](snake_blockbreaker/README.md) | Snake and block breaker game combo |
| [snake_game_2led](snake_game_2led/README.md) | Snake game on 2 LED matrices |

For build and flash instructions, please refer to the [root README](../../README.md).
